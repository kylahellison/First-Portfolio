Hi, I'm Kylah!

I started teaching myself how to code websites back in the days when AOL Instant Messenger, MySpace, and flashy GIFs were some of the hottest trends on the internet. I'd spend hours analyzing code using "View Source" to figure out how everything worked.

When I went off to college, I took my studies in a different direction. Years later, I found myself wishing I had stuck with coding.

I started researching ways that I could get back into coding, and stumbled across teamtreehouse.com, an online technology school that offers courses in web design, web development, and mobile development. I was accepted into their Front End Web Development TechDegree beta program, which is designed to teach you the fundamentals of front-end web development, including HTML, CSS, JavaScript, accessibility and user experience.

The projects you see in my portfolio were generated from this curriculum, and showcase all of the things I learned throughout my TechDegree experience. On the 'Treehouse' page of my site, you can see all of the types of courses I've taken through Treehouse and where I've focused my learning.

I am currently looking for work to build up my portfolio with additional projects while I continue taking courses on Treehouse to make sure I'm always up to date with the latest in web design.

Feel free to get a hold of me using the "Contact Me" page if you have any questions or would like to discuss a potential project.
